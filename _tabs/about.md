# Hi, I'm Ayman.

I've always liked taking things apart just to see how the gears, wires, and code actually talk to each other. That habit eventually turned into a serious interest in mechatronics. 

I just finished high school and I'll be starting university this year. As I started working on more complex projects, I noticed I was accumulating a lot of messy wiring diagrams, half-written code, and random notes. I needed a place to organize it all, which is why I set up [**aymano.com**](https://aymano.com).

I want to be completely upfront—I’m not an engineer yet. I’m a student. This website and my [YouTube channel](https://youtube.com/@aymanalsakkaf) are basically just my open notebook. I’m posting my projects exactly as they happen: the parts that work perfectly, and the debugging nightmares that keep me up late. 

I'm putting this out there hoping that seeing my trial-and-error process might save another beginner a few headaches. Thanks for reading, and let's build something.
